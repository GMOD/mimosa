mimosa by GMOD

Miniature Model Organism Sequence Aligner

Mimosa allows researchers to run sequence alignment programs on nucleotides or proteins, and request sequences from various sequence databases, all from a friendly web interface.

Dependencies

Perl 5, Moose, Catalyst

Install

git clone git://github.com/GMOD/mimosa.git cd mimosa perl Build.PL ./Build --installdeps # install necessary dependencies ./Build
