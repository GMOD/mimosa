mimosa by GMOD

Miniature Model Organism Sequence Aligner

Mimosa allows researchers to run sequence alignment programs on nucleotides or proteins, and request sequences from various sequence databases, all from a friendly web interface.

Dependencies

BLAST, Perl 5, Moose, Catalyst, BioPerl, Bio::GMOD::Blast::Graph, Bio::BLAST, Bio::Chado::Schema and many more awesome CPAN modules.

Install

    git clone git://github.com/GMOD/mimosa.git
    cd mimosa
    perl Build.PL
    ./Build --installdeps # install necessary dependencies
    ./Build
If you have cpanminus installed (it is highly recommended), you can use that to install dependencies as well: 
    git clone git://github.com/GMOD/mimosa.git
    cd mimosa
    cpanm --installdeps . # install necessary dependencies
    perl Build.PL
    ./Build
